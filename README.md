# FastAPI Learning Project

This repository contains my learning journey with the FastAPI framework.

## Topics Covered
- FastAPI basics
- Path parameters
- Query parameters
- Pydantic BaseModel
- Routers
- API testing using Swagger

## Running the project

Create virtual environment:

python -m venv venv

Activate:

source venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Run server:

uvicorn main:app --reload

API Docs:

http://127.0.0.1:8000/docs
